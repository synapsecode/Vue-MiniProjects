sin, cos, tan, inverse trignometry, log10, naturalLog, e^x, 10^x, squaring, sqrt, cuberoot, exponents, nth-roots, percentage, reciprocals, abs, factorial, E (denoting number of zeros) and the basic operations(+, -, √∑, x) along with some constants such as ùúã and e.
